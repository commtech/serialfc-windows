## Disconnect
The Windows [`CloseHandle`](http://msdn.microsoft.com/en-us/library/windows/apps/ms724211.aspx) is used to disconnect from the port.


###### Examples
```c
#include <Windows.h>
...

CloseHandle(h);
```
